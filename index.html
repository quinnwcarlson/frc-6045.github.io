Step 1 · Get your computer ready

Download Important Applications

WPILib VS Code

The software 6045 uses to edit and deploy robot code.

Sartell school MacBook: install JDK and Xcode first (both on Self Service). Xcode needs administrator credentials.
Download

Elastic

Monitors robot status — speed, Limelight view, and encoder position. Also used to select autos.

Open it from WPILib VS Code

GitHub Desktop

How 6045 accesses code across multiple computers. Not available for Linux.

Download Why we use it

FRC Game Tools

National Instruments' package including the Driver Station — necessary to control the robot. Windows only.

Download

REV Hardware Client

What 6045 uses to assign CAN IDs to motors.

As of February 2026, this can't run on Sartell school MacBooks — their OS is not recent enough.
Download

PathPlanner

The software 6045 uses to make autos. Click the heading with the green "Latest" tag, then scroll down and pick the file matching your OS.

School MacBook: install the ZIP, then run xattr -d com.apple.quarantine file_path in the terminal. Drag the installer into the terminal to get its path fast.
Download

iTerm2 (Mac only)

A replacement terminal for Apple computers. Only needed on Sartell school MacBooks, where the regular Terminal app can't be opened.

Download
