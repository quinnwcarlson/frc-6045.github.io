---
title: "How to Deploy Code"
order: 4
---

<ol>
    <li><a href="howtoconnecttotherobot.html">Connect to the robot</a>.</li>
    <li>Open <a href="https://docs.wpilib.org/en/stable/docs/zero-to-robot/step-2/wpilib-setup.html" target="_blank" rel="noopener">WPILib VS Code</a>.</li>
    <li>Click the three dots in the top right corner of VS Code (circled in orange).
      <img src="assets/images/howtodeploycode/1.png" alt="VS Code window with the three-dot menu circled in orange">
    </li>
    <li>Click the button that says <strong>Deploy Robot Code</strong> (highlighted in orange).</li>
  </ol>

  <p>After this, a message will appear near the bottom of VS Code:</p>

  <div class="callout warn"><strong>Red</strong> means the deployment failed — most likely, the computer was not connected to the robot.</div>
  <img src="assets/images/howtodeploycode/2.png" alt="VS Code showing a red failed-deployment message">

  <div class="callout ok"><strong>Green</strong> means the deployment succeeded.</div>
  <img src="assets/images/howtodeploycode/3.png" alt="VS Code showing a green successful-deployment message">
